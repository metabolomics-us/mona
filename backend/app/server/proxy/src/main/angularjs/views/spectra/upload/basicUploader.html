MoNA requires that every spectrum has an associated compound. Please enter an InChI, InChIKey, SMILES, compound name, a MOL/SDF file, or draw a structure yourself. Then, click on "Update Compound" to automatically pull the remaining fields from the Chemical Translation Service. Please note that some names or InChIKeys do not exist in compound databases, and so the CTS may not find anything - in this case, using a full structure such as InChI, SMILES or MOL is preferred.

Please note that using this tool will overwrite any existing MOL file, InCHI code or InChIKey.
