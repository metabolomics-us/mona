Welcome to MoNA!

MassBank of North America (MoNA) is a metadata-centric, auto-curating repository designed for efficient storage and querying of mass spectral records. It intends to serve as a the framework for a centralized, collaborative database of metabolite mass spectra, metadata, and associated compounds. MoNA currently contains {{totalCount | number:'1.0-0'}} mass spectral records from experimental and in-silico libraries, as well as user contributions.

News

MoNA Overhaul
September 15th, 2025

MoNA has received a major overhaul. Issues such as uploading spectra are finally fixed, and almost every piece of the frontend has been updated with functionality and user experience in mind. If you encounter any issues or have any suggestions, please report them to the issue tracker. For a more complete list of changes, see the release notes.

MoNA Backend Updates
January 1st, 2023

MoNA has undergone substantial changes to its backend services. Frontend features work as they did before with an exception of the RSQL query system which has been changed to the Spring Filtering system. For more detailed notes of the code changes, click here. To see new schema changes, see the updated Swagger API Docs.

Ability to Delete Your Submitted Spectra
March 4th, 2022

A new feature has been introduced that allows signed in users to delete spectra that matches their submitter ID. This feature is embedded into the spectra browser where you will see a new "My Submitted Spectra" button that can query for your spectra, as well as a "Mark for Deletion" checkbox around your submitted spectra. Once selected, just hit the "Batch Delete" button and remove spectra that needs to be removed.
